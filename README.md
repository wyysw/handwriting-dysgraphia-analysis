# Handwriting Dysgraphia Analysis

This repository contains the implementation for the undergraduate thesis:

**Algorithm Design and Implementation for Evaluating Handwriting Interaction Data in Children with Dysgraphia**

## Research Objectives
- Analyze handwriting interaction data (trajectory, velocity, pressure, timing)
- Design feature extraction algorithms
- Evaluate handwriting performance related to dysgraphia

## Methods
- Signal preprocessing
- Feature extraction
- Machine learning–based evaluation

## Technologies
- Python
